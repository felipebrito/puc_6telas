=============================================================
GUIA DEFINITIVO: INSTALAÇÃO MAC MINIS - PUC 6 TELAS
=============================================================

### Fase 1: Preparação do macOS (Em ambos os Mac Minis)
Antes de instalar qualquer software, precisamos garantir que o Mac não vai dormir, desligar a tela ou pedir senha no meio da exposição.

1. Configurações de Energia e Tela:
   - Vá em Ajustes do Sistema > Telas: Desative protetores de tela e bloqueio automático.
   - Vá em Ajustes do Sistema > Tela Bloqueada: Defina "Desligar a tela" e "Exigir senha" para Nunca.
   
2. Ocultar Interface (Kiosk Mode):
   - Vá em Ajustes do Sistema > Mesa e Dock: Marque para ocultar o Dock automaticamente.
   - Na mesma tela, escolha ocultar a Barra de Menus automaticamente.

3. Rede:
   - No momento os Macs estão configurados para "Auto-Discovery".
   - Basta que ambos estejam na mesma rede (via cabo ou Wi-Fi) para o Slave encontrar o Master automaticamente. 
   - Quando for a hora do projeto final, você poderá ir no Ajustes do Sistema > Rede e fixar o IP para 192.168.3.11 e 192.168.3.12. O auto-discovery continuará funcionando sem problemas mesmo com os IPs novos.

---

### Fase 2: Instalação dos Programas Base (Em ambos os Mac Minis)
Abra o aplicativo Terminal (fica em Aplicativos > Utilitários).

1. Instalar o Homebrew (Gerenciador de pacotes da Apple):
   Copie e cole o comando abaixo no terminal e dê Enter (ele vai pedir a senha do usuário do Mac):
   
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   
   *(Atenção: Ao final da instalação, o Terminal pode pedir para você rodar dois comandos para adicionar o Brew no PATH. Apenas copie e cole as linhas que ele sugerir no final da tela e dê Enter).*

2. Instalar o Player de Vídeo (mpv), o Node.js e o Git:
   Ainda no Terminal, rode:
   
   brew install mpv node git

---

### Fase 3: Baixando e Configurando o Projeto (Em ambos os Mac Minis)
Abra o Terminal e rode os comandos abaixo para baixar o código e as dependências.

1. Clonar o Repositório:
   cd ~/Desktop
   git clone https://github.com/felipebrito/puc_6telas.git
   cd puc_6telas

2. Instalar as dependências de Código e o PM2 (o PM2 mantém os vídeos rodando em 2o plano e religa se o PC reiniciar):
   npm install
   npm install -g pm2

3. Configurar os vídeos:
   Abra o arquivo `ecosystem.config.js` (com o Editor de Texto do Mac ou VSCode).
   - Em VIDEO_1_PATH, VIDEO_2_PATH e VIDEO_3_PATH, coloque o caminho real dos vídeos daquela máquina.
   - Não precisa se preocupar com o MASTER_IP, ele agora está configurado como "auto" e encontrará o Master na rede sozinho.

---

### Fase 4: Execução e Automação (Iniciando o Show)

>> No Mac Mini 1 (Que será o Controlador e tocará as telas 1, 2 e 3):
1. No terminal, dentro da pasta puc_6telas, rode:
   pm2 start ecosystem.config.js
   *(Isso inicia o Master e o Slave deste computador).*

>> No Mac Mini 2 (Que apenas tocará as telas 4, 5 e 6):
1. No terminal, dentro da pasta puc_6telas, rode:
   pm2 start ecosystem.config.js --only slave
   *(Isso inicia apenas o tocador)*

>> Salvando para inicializar com o computador (Nas duas máquinas):
Para garantir que os vídeos abram assim que alguém ligar o computador (se faltar luz):
1. Rode no terminal:
   pm2 startup
2. O terminal vai gerar um comando longo. Copie esse comando inteiro, cole no próprio terminal e dê Enter.
3. Agora rode:
   pm2 save

---

### Fase 5: Operação (Como dar Play/Pause/Seek)
Como a aplicação agora roda em 2º plano, sempre que quiser dar Play ou Pause em todos os vídeos simultaneamente:

1. No Mac Mini 1 (O Controlador), abra um novo terminal e rode:
   node /Users/seu_usuario/Desktop/puc_6telas/master.js

2. Digite "play" e dê Enter. As duas máquinas tocarão os vídeos sincronizadas instantaneamente.
   Comandos suportados: play, pause, seek <segundos>, load.
